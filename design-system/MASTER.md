# Altis Groep — Design System MASTER

**Direction:** Industrial financial terminal — Bloomberg meets Dutch roofing precision.  
**Audience:** CFO + Opco MD at Altis Groep PE portfolio.  
**Stack:** React 18 + Tailwind v4 + Recharts + Lucide.

## Typography
- **Display:** DM Serif Display — headings, brand
- **Body:** IBM Plex Sans — labels, UI copy
- **Data:** JetBrains Mono — ALL euro amounts, ratios, weeks

## Color tokens
- `--bg-primary` #0a0c10
- `--bg-secondary` #12151c  
- `--bg-elevated` #1a1f2b
- `--accent-copper` #c9a227 (brand highlight)
- `--accent-blue` #3b82f6 (interactive)
- `--accent-green` #22c55e (positive / on track)
- `--accent-amber` #eab308 (warning)
- `--accent-red` #ef4444 (covenant / delayed)

## Layout
- Sidebar 240px (role + scenario)
- Max content width: 1400px
- 8px spacing rhythm
- Cards: 1px border white/8%, subtle inner glow

## Accessibility
- Contrast ≥4.5:1 for body text
- Focus rings: 2px accent-blue
- Touch targets ≥44px
- prefers-reduced-motion respected

## Terminology (never deviate)
WIP Exposure, Covenant Headroom, Milestone Billing, Opco, 13-Week Rolling Forecast, Driver Model, Materials Outflows
